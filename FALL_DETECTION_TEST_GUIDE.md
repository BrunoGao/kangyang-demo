## 🔧 故障排除

### 常见问题

**Q1: API服务无法启动**
```bash
# 检查端口占用
lsof -i :6000

# 重新启动服务
cd ai-detection
source venv/bin/activate
python fall_demo_api.py
```

**Q2: 视频加载失败**
```bash
# 检查视频文件
ls -la /Users/bg/work/codes/kangyang-demo/mp4/falldown.mp4

# 检查权限
chmod 644 mp4/falldown.mp4
```

**Q3: 前端无法连接后端**
- 检查防火墙设置
- 确认API服务正在运行
- 验证端口配置正确

**Q4: 检测结果不准确**
- 调整置信度阈值
- 检查视频质量
- 验证测试环境配置
